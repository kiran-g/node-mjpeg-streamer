# node-mjpeg-streamer
 Mjpeg streamer with v4l2 as camera interface

## Usage

```shell
'Usage: node mjpeg-streamer

  -p, --port=ARG    Port
  -d, --device=ARG  V4L2 Device number. 0 for "/dev/video0"
  -h, --help        display this help
  -v, --version     show version
```
